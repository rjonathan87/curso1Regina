🔍 Unidad 1 — Información Digital

Unidad 1
Bit
La unidad mínima de información digital. Solo puede ser 0 o 1. El nombre viene de Binary digiT.
Unidad 1
Byte
Conjunto de 8 bits. Representa un carácter (letra, número o símbolo). Es la unidad básica de almacenamiento.
Unidad 1
Kilobyte (KB)
1,024 bytes. Equivale aproximadamente a una página de texto sin formato.
Unidad 1
Megabyte (MB)
1,024 KB. Una foto en alta calidad ocupa ~3–5 MB. Una canción MP3, ~4 MB.
Unidad 1
Gigabyte (GB)
1,024 MB. Una película HD ocupa ~1–4 GB. Los celulares suelen tener 64–256 GB de almacenamiento.
Unidad 1
Malware
Software malicioso diseñado para dañar, robar o tomar control de sistemas sin permiso del usuario.
Unidad 1
Phishing
Técnica de engaño digital donde atacantes se hacen pasar por empresas legítimas para robar datos personales.
Unidad 1
Antivirus
Programa que detecta, previene y elimina malware. Windows Defender viene incluido en Windows 10/11.
Unidad 1
Nube (Cloud)
Almacenamiento y servicios en servidores remotos accesibles por internet. Ejemplos: Google Drive, iCloud, Dropbox.

🖥️ Unidad 2 — Procesamiento Digital

Unidad 2
Hardware
Los componentes físicos y tangibles de una computadora: CPU, RAM, disco duro, teclado, pantalla.
Unidad 2
Software
Conjunto de programas e instrucciones que le dicen al hardware qué hacer. No es físico, es lógico.
Unidad 2
CPU (Procesador)
Central Processing Unit. El "cerebro" de la computadora que ejecuta instrucciones. Su velocidad se mide en GHz.
Unidad 2
RAM
Random Access Memory. Memoria temporal que almacena lo que estás usando ahora. Se borra al apagar. Más RAM = más multitarea.
Unidad 2
Disco Duro / SSD
Almacenamiento permanente. HDD (mecánico, más lento) vs SSD (sólido, mucho más rápido). Los datos persisten al apagar.
Unidad 2
Sistema Operativo (SO)
Software base que gestiona el hardware y permite que otros programas funcionen. Windows, macOS, Linux, Android.
Unidad 2
Periférico
Dispositivo externo conectado a la computadora. De entrada (teclado, mouse) o salida (impresora, pantalla).
Unidad 2
Software Libre
Programas cuyo código fuente es público, gratuito y modificable. Ejemplos: LibreOffice, VLC, Linux.

⚙️ Unidad 3 — Algoritmos y Programación

Unidad 3
Algoritmo
Secuencia finita, ordenada y precisa de pasos para resolver un problema. Cualquier receta de cocina es un algoritmo.
Unidad 3
Diagrama de flujo
Representación visual de un algoritmo usando símbolos estándar: óvalos (inicio/fin), rectángulos (acciones), rombos (decisiones).
Unidad 3
Pseudocódigo
Descripción informal de un algoritmo usando lenguaje natural estructurado (español). No tiene sintaxis rígida de programación.
Unidad 3
Variable
Espacio con nombre que almacena un dato que puede cambiar durante la ejecución. Como una caja con etiqueta.
Unidad 3
Condicional (Si/Sino)
Estructura que ejecuta diferentes pasos según si una condición es verdadera o falsa. "Si llueve, lleva paraguas; si no, no."
Unidad 3
Bucle (Loop)
Estructura que repite un bloque de instrucciones hasta que se cumple una condición. "Mientras haya ropa sucia, lavar."
Unidad 3
Scratch
Plataforma de programación visual (bloques) creada por el MIT. Ideal para aprender lógica sin preocuparse por sintaxis.
Unidad 3
Bug / Error
Fallo en un programa que produce resultados incorrectos. "Debugging" es el proceso de encontrar y corregir bugs.
Unidad 3
Paradigma de programación
Estilo o enfoque para estructurar el código. Los más comunes: imperativo, orientado a objetos (Python, Java), funcional.
Unidad 3
Pensamiento Computacional
Habilidad para descomponer problemas complejos en pasos simples que una computadora (o persona) puede ejecutar.
